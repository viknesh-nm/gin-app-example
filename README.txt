Requirements:

    Go V1.7 --> Gin Framework
    MySql (cmd line or workspace)

Installation:

    Go :
        Download GO from the link(https://golang.org/dl/)

    MySql:
        Download MySQL from the link(https://dev.mysql.com/downloads/mysql/)

    Since We are using Gin Framework click on the official site (https://gin-gonic.github.io/gin/)

    -- Open the command prompt
        -> go get github.com/gin-gonic/gin
            - The dependencies of gin will be downloaded


Procedure to run the application:

    1. go build

    2. admin-panel.exe (windows) or ./admin-panel(ubuntu)
